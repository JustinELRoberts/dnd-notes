# Death Saving Throw

_Source: Player's Handbook (2024) p. 365. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)_

A [player character](/Mechanics/CLI/variant-rules/player-character-xphb.md) must make a Death Saving Throw (also called a Death Save) if they start their turn with 0 [Hit Points](/Mechanics/CLI/variant-rules/hit-points-xphb.md).
