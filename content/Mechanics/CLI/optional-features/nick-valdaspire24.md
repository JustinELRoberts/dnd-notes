# Nick

_**Prerequisites**_ Level 2 Craftsman, Light Property

When you make the extra attack of the [Light](/Mechanics/CLI/item-properties.md#Light) property, you can make it as part of the [Attack](/Mechanics/CLI/actions.md#Attack) action instead of as a [Bonus Action](/Mechanics/CLI/variant-rules/bonus-action-xphb.md). You can make this extra attack only once per turn.
